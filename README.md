## **Installation**

**Cloning the repo**

```bash
git clone https://github.com/Stable-crypto/stable-landing-page/
```

**Installing the dependencies**

```bash
yarn install
```

**Starting the localhost-server**

```bash
yarn dev
```

**Creating the build**

```bash
yarn build
```

**Starting the build-version**

```bash
yarn start
```
